# Agentic Agora (아젠틱 아고라)

이 프로젝트는 다양한 AI 에이전트(Antigravity, GEMINI CLI 등)를 체계적으로 조율하고 활용하여 더 큰 프로젝트를 구성하기 위한 **에이전트 중심 생산성 프레임워크**입니다.

## 🧭 프로젝트 지도 (Map)

본 저장소는 단일 진실 공급원(SSOT) 원칙에 따라 각종 철학과 결정, 기술 문서를 명확히 분리하여 관리합니다. 저장소를 파악하기 위해 아래 링크들을 순서대로 확인해 주세요.

- [VISIONS.md](VISIONS.md) : "이 시스템을 왜 만드는가?" 프로젝트의 존재 목적과 비전, 우리가 나눈 철학적 대화 기록
- [ADR-INDEX.md](ADR-INDEX.md) : 아키텍처 및 핵심적인 시스템 구조 결정 상태 인덱스
- [docs/INDEX.md](docs/INDEX.md) : 기술 문서(매뉴얼), 로드맵, 용어집 등 실무 문서를 연결하는 허브 진입점

## 🛠 주요 디렉토리 구조
- `docs/` : 에이전트 가이드, 용어집 등 상세 기술/기획 문서가 포함되어 있습니다.
