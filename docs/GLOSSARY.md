# 용어 사전 (Glossary)

본 프로젝트 전반에 걸쳐 사용되는 도메인 특화 용어 및 개념들을 단일 진실 공급원(SSOT) 원칙에 따라 정의합니다.

| 용어 | 영문 | 의미 요약 |
| :--- | :--- | :--- |
| **에이전트 툴** | Agent Tool | `Google Antigravity`, `GEMINI CLI` 등 AI 기반의 작업을 수행하고 코딩을 돕는 강력한 도구. |
| **스킬** | Skill | 에이전트가 특정 목적(예: 인프라 구성)을 수행하기 위해 부여받는 재사용 가능한 지식 패키지. 보통 `SKILL.md`를 사용. |
| **트랙** | Track | 하나의 명확한 단기 목표를 달성하기 위해 생성되는 작업의 단위. `tracks/` 내부에서 독립된 상태(`plan.md`)로 관리됨. |
| **SSOT** | Single Source of Truth | 정보의 파편화를 막기 위해, 데이터나 문서의 원천은 오직 하나의 고유한 파일로만 관리되어야 한다는 단일 진실 공급원 원칙. |
| **ADR** | Architecture Decision Records | 프로젝트 전체의 구조, 시스템 등 중요한 방향과 아키텍처 결정을 기록해둔 문서. |
| **MCP** | Model Context Protocol | AI 에이전트가 외부 데이터 소스나 툴과 데이터를 주고받기 위한 통신 표준 규격('AI의 USB-C'). |
| **서브에이전트** | Subagent | 메인 에이전트의 지시를 받아 고립된 컨텍스트 내에서 단일 작업(리서치 등)을 수행하는 보조 에이전트. |
| **멀티에이전트** | Multi-agent | 두 개 이상의 AI 에이전트가 협력하여 단일 에이전트의 한계를 넘어 복잡한 문제를 해결하는 운영 형태. |
| **에이전트 팀** | Agent Team | 개별 에이전트들이 직접 통신하고 작업 목록을 자율적으로 공유하며 실시간 병렬 협력하는 고급 아키텍처. |
